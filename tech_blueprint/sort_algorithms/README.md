## Algorithms

1. Radix - O(n)
2. STD - O(n log n)
3. Shell Sort - O(n log n)
4. Selection - O(n^2)
5. Quick sort - O(n log n)
6. Bubble sort - O((n^2) / 2)
7. Insertion sort - O((n^2) / 4)
